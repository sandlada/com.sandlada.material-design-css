# Typography

Typography includes 30 styles, including 15 classic styles and 15 emphasized styles. These 15 styles are composed of 5 parts.:

- Display
- Headline
- Title
- Body
- Label

Each part has three sizes:

- Large
- Medium
- Small

Each of the 30 styles has a single token that captures all the default properties. Tokens are separated into the baseline and emphasized sets. Each axis and property, such as font, line height, size, tracking, and weight, also has an individual token for greater customization. 

For a detailed description of M3 Typography, see [Definition of M3 Typography](https://m3.material.io/styles/typography/type-scale-tokens).

## Import

### For CSS

_For CSS, we don't provide more detailed class names like tailindcss._

```css
@import "@sandlada/material-design-css/typography/typography-utilities.css";
```

### For TailwindCSS v4

_For CSS, we don't provide more detailed class names like tailindcss._

```css
@import "@sandlada/material-design-css/typography/tailwind-theme.css";
@import "@sandlada/material-design-css/typography/tailwind-utilities.css";
```
